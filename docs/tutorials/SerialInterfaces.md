# List of serial connection options

There are several options for serial communication with the ESP32. **Here are the tested ones:**

- JavaScript with the [Web Serial API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Serial_API) as used in the [Photobooth Application](./code/javascript/readme.md)
- Python with the [`pyserial`](https://pythonhosted.org/pyserial/) library as used in the [Photobooth Application](./code/python/readme.md)

**Other options include:**

- JavaScript in p5.js with the [p5.serialport library](https://github.com/p5-serial/p5.serialport)
- Scratch with [ScratchSerialExtension](https://github.com/amandaghassaei/ScratchSerialExtension)
- Godot Engine with the [Godot Serial Plugin](https://godotengine.org/asset-library/asset/4175)

Feel free to explore these options and choose the one that best fits your project needs!
